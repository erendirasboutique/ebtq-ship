# Erendira's Boutique EasyPost Shipping Studio v3

## Vercel environment variables

Required:
- EASYPOST_API_KEY
- NEXT_PUBLIC_SUPABASE_URL
- SUPABASE_SERVICE_ROLE_KEY
- SHIPPING_PORTAL_PASSWORD
- SHIPPING_SESSION_SECRET

Optional:
- SHIP_FROM_PHONE
- SHIP_FROM_EMAIL

## Supabase
Run `sql/setup.sql` in Supabase SQL Editor before using the portal.

## Notes
Do not upload `node_modules`, `.next`, `.env`, or `.env.local` to GitHub.
